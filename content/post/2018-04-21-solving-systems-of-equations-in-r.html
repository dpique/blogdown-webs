Imagine that we have a system of equations with 3 variables and 3 unknowns. It looks as follows:

Below is some code for how to solve this system of equations in R:

A <- array(c(3,2,-1,2,-2,0.5,-1,4,-1), dim = c(3,3))
b <- c(1,-2,0)
A
##      [,1] [,2] [,3]
## [1,]    3  2.0   -1
## [2,]    2 -2.0    4
## [3,]   -1  0.5   -1
x <- solve(a = A, b = b)
Ax <- A %*% x %>% 
  as.numeric %>% 
  round(10)

all.equal(Ax, b) #are the two equal?
## [1] TRUE
invA <- solve(A)
invA %*% b %>% 
  as.numeric %>% 
  all.equal(x)
## [1] TRUE
