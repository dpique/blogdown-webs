## Bioinformatics and genomics software

1. **Aneuvis** is a web application for analyzing chromosomal variation in single cells built with Shiny. ## Data-containing software packages

1. **knudson** is an R data package that contains a machine-readable form of Table 1 from Alfred Knudson's classic paper describing the two-hit cancer hypothesis (1971, PNAS). An vignette attempts to recreate Figure 1 from the data in Table 1. Code: <span style="vertical-align: -6px;"> <a href="https://github.com/dpique/knudson" target="_blank"> <img alt = 'github' width='30' src='https://img.icons8.com/windows/144/000000/github.png'/> </a> </span> [add CRAN logo]

1. [hotdog]() - R data package containing the placing of contestants from Nathan's Hot Dog Eating contest from 2002-2019.
